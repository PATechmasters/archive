Hi Techmasters,

Some follow up from today’s meeting:

In real CTF contests, you can and often have to use the internet. Even if you’ve been doing this for years. The point is agility, not memorization. Use any tools and resources you want; the only thing you’re not supposed to do is ask a human for help.

TMCTF is just for practice. This means scores don’t matter. More importantly, it means you can and should ask humans for help. Ask your peers, ask us, email us, whatever.

Don’t give up; look up solutions for TMCTF problems. All of them are real problems from past CTFs, so if you search for the problem name and CTF name (in parenthesis at the end of each problem), you’ll find that people have posted solutions.

This isn’t cheating; it’s learning. I couldn’t have done these problems when I was starting out. You need a base, and looking up solutions and solving the problems by yourself afterward is an awesome way to build it.

The server problems from meeting one are still up and running! Let us know if you’re facing issues or just don’t know where to start on a particular level.

Most of all, remember this: We don’t care whether you got it right the first time. We care whether you get it now, and all of us here are here to make that happen.

Happy hacking,
